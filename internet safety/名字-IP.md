# 名字-IP

- 定义，IP 地址（或互联网协议地址）可用于在一段时间内识别网络上的主机，之后该 IP 地址可以关联到另一个设备而无需更改。
- 构成，IP地址是一组数字， 分为四个八位字节192.168.1.1
- 由来，ip地址分配和子网划分计算

## 特点

- IP 地址在不同的设备上可能会有所不同，但同一个网络中的 IP 地址不能同时处于多个活动状态
- IP 地址遵循一套称为协议的标准。这些协议是网络的基础，它们强制许多设备使用同一种语言进行通信
- 设备既可以位于私有网络，也可以位于公共网络。设备所在位置决定了它们拥有哪种类型的 IP 地址：公共 IP 地址或私有 IP 地址。
- 这两个设备可以使用各自的私有IP地址进行通信。但是，从这两个设备发送到互联网的任何数据都将使用同一个公共IP地址进行标识。公共IP地址由您的互联网服务提供商（ISP）按月收费（费用包含在您的账单中！）
- IPv4，IPv6，IPv6解决IP地址不够的问题并且有两个优点：
- 支持高达 2^128 个 IP 地址（超过 340 万亿个）
- 由于采用了新的方法，效率更高

## 指纹-媒体访问控制地址（MAC） 类似序列号

- 定义，网络上的所有设备都具有物理网络接口，即位于设备主板上的微芯片板。该网络接口在出厂时会被分配一个唯一的地址，称为MAC（媒体访问控制）地址。MAC 地址是一个12位十六进制数（计算机中用于表示数字的 16 进制计数系统），以冒号分隔成两部分。这些冒号用作分隔符。例如，a4:c3:f0:85:ac:2d。前六个字符代表网络接口的制造商，后六个字符是一个唯一的数字。
- MAC地址欺骗，MAC地址可以伪造

## ping工具（可以直接在cmd中使用）

- 语法ping IP address or website URL
- Ping是我们可用的最基本的网络工具之一。Ping 使用ICMP（互联网控制消息协议）数据包来确定设备之间连接的性能，例如，连接是否存在或是否可靠。可以对网络上的设备（例如您的家庭网络或网站等资源）执行 Ping 操作。此工具易于使用，并且已预装在操作系统 (OS) 中，例如：Linux以及 Windows 系统。
